# Security Policy

This repository is an educational Solidity starter project.

## Important warnings

- Do not use the contract in production without a professional audit.
- Do not commit `.env`, private keys, seed phrases, or real RPC credentials.
- Use a separate test wallet for deployments.
- Start on local Hardhat network or Sepolia testnet before using any mainnet.

## Reporting issues

If you find a bug, open a GitHub issue with:

1. Contract/function name
2. Steps to reproduce
3. Expected behavior
4. Actual behavior
